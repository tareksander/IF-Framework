updateAvailable

updateAvailable: Readable<boolean>

Whether an update is available, downloaded and will be applied on a page refresh. A value of false means there is no update available, there was no check yet, config.json or service.js is missing in the directory of the story HTML file, or the browser doesn't support service workers.
